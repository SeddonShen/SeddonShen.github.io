# 本地开发环境配置指南

## 前置要求

1. **Hugo Extended** - 已安装 ✅ (v0.136.5)
2. **Go** - 用于支持 Hugo modules（已安装）

## 快速开始

### 方法一：使用启动脚本（推荐）

```bash
./dev.sh
```

### 方法二：直接使用 Hugo 命令

```bash
# 基本启动（仅本地访问）
hugo server

# 完整功能启动（包含草稿，允许外部访问）
hugo server -D --bind 0.0.0.0

# 自动导航到更改的文件
hugo server -D --bind 0.0.0.0 --navigateToChanged
```

## 常用命令

### 启动开发服务器
```bash
hugo server
```
- 默认地址：http://localhost:1313
- `-D` 参数：包含草稿内容
- `--bind 0.0.0.0`：允许局域网内其他设备访问

### 更新主题
```bash
hugo mod get -u github.com/geekifan/zero-academic-page
hugo mod tidy
```

### 构建静态网站
```bash
hugo
```
生成的文件在 `public/` 目录

### 清理缓存
```bash
hugo --cleanDestinationDir
```

## 项目结构

- `content/` - 网站内容（Markdown 文件）
- `assets/` - 静态资源（图片、CSS、JS 等）
- `hugo.toml` - Hugo 配置文件
- `go.mod` / `go.sum` - Go modules 依赖管理

## 开发流程

1. 启动开发服务器：`./dev.sh` 或 `hugo server`
2. 在浏览器中打开 http://localhost:1313
3. 编辑 `content/` 目录下的 Markdown 文件
4. 保存后，Hugo 会自动重新生成页面（热重载）
5. 刷新浏览器查看更改

## 故障排除

### 如果遇到模块下载问题
```bash
hugo mod clean
hugo mod get -u
hugo mod tidy
```

### 如果端口被占用
```bash
hugo server --port 1314
```

### 查看详细日志
```bash
hugo server --verbose
```
